# Data Structures and Algorithms - Java Lab Experiments Programs

This repository contains Java implementations of fundamental Data Structures and Algorithms (DSA) problems. It covers a wide range of topics including Arrays, Recursion, Sorting, Searching, Linked Lists, Stacks, Queues, Trees, and Graphs.

## üõ† Prerequisites

*   **Java Development Kit (JDK):** Version 8 or higher.
*   **IDE (Optional):** IntelliJ IDEA, Eclipse, VS Code, or simple Text Editor/Command Line.

## üìù Concepts Covered

*   **Array Manipulation:** Insertion, Deletion, Matrix Math.
*   **Recursion:** Base cases, Stack frames, Iterative vs Recursive approaches.
*   **Sorting Algorithms:** O(n^2) and O(n log n) algorithms.
*   **Searching:** Linear vs Binary Search.
*   **Data Structures:** Linked Lists (Singly, Doubly, Circular), Stacks, Queues (Circular).
*   **Non-Linear Data Structures:** Trees (BST, Traversals) and Graphs (BFS, DFS).
